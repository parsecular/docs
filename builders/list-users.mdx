title: "List Users"
description: "List all end-users in your builder account."
---

`GET /api/v1/builder/users`

## Parameters

<ParamField query="limit" type="number" default="50">
  Items per page (1â€“100).
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor from `next_cursor`.
</ParamField>

## Response

<ResponseField name="users" type="BuilderUserResponse[]">
  Array of user objects (same shape as the [create user response](/builders/create-user#response)).
</ResponseField>

<ResponseField name="next_cursor" type="string | null">
  Pass as `cursor` to fetch the next page. Omitted on the last page.
</ResponseField>

<ResponseField name="count" type="number">
  Number of items in this response.
</ResponseField>
