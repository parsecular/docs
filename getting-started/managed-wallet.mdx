## Overview

With managed credentials, Parsec handles everything: wallet creation, credential
derivation, token approvals, and transaction signing. You don't manage private keys
or hold gas tokens.

**Best for:** Traders who want the fastest setup, and builders onboarding users
who don't have existing wallets.

```
Your App ───► POST /onboard { mode: "managed" } ───► Parsec creates wallet + credentials
         ───► POST /orders                        ───► Parsec signs and routes
```

## Step 1: Get your API key

Sign up at [parsecapi.com](https://parsecapi.com) → **Settings** → copy your API key (`pk_live_...`).

## Step 2: Onboard with managed mode

A single call creates your wallet, derives CLOB credentials, and sets token approvals:

<CodeGroup>
```bash curl
curl -X POST "https://api.parsecapi.com/api/v1/onboard" \
  -H "X-API-Key: pk_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "exchange": "polymarket",
    "mode": "managed"
  }'
```
```typescript TypeScript
import ParsecAPI from 'parsec-api';

const client = new ParsecAPI({ apiKey: 'pk_live_YOUR_KEY' });

const result = await client.onboard.create({
  exchange: 'polymarket',
  mode: 'managed',
});
console.log(result.wallet_address);
```
```python Python
from parsec_api import ParsecAPI

client = ParsecAPI(api_key="pk_live_YOUR_KEY")

result = client.onboard.create(
    exchange="polymarket",
    mode="managed",
)
print(result.wallet_address)
```
</CodeGroup>

<ResponseExample>
```json
{
  "exchange": "polymarket",
  "mode": "managed",
  "status": "complete",
  "wallet_address": "0x1234567890abcdef1234567890abcdef12345678",
  "linked_exchanges": ["polymarket"],
  "steps_completed": ["wallet_eoa_created", "exchange_linked"]
}
```
</ResponseExample>

<Info>
Idempotent — calling this again skips already-completed steps and returns the same result.
</Info>

### Safe wallet (optional)

For Gnosis Safe wallets, pass `wallet_type: "safe"`:

```bash
curl -X POST "https://api.parsecapi.com/api/v1/onboard" \
  -H "X-API-Key: pk_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "exchange": "polymarket",
    "mode": "managed",
    "wallet_type": "safe"
  }'
```

## Step 3: Fund your wallet

Send **USDC** to your `wallet_address` on **Polygon**. Gas fees are sponsored — no POL needed.

## Step 4: Trade

<CodeGroup>
```bash curl
curl -X POST "https://api.parsecapi.com/api/v1/orders?exchange=polymarket" \
  -H "X-API-Key: pk_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "market_id": "0x1234abcd",
    "outcome": "Yes",
    "side": "buy",
    "price": 0.55,
    "size": 100
  }'
```
```typescript TypeScript
const order = await client.orders.create({
  exchange: 'polymarket',
  market_id: '0x1234abcd',
  outcome: 'Yes',
  side: 'buy',
  price: 0.55,
  size: 100,
});
console.log(order.id, order.status);
```
```python Python
order = client.orders.create(
    exchange="polymarket",
    market_id="0x1234abcd",
    outcome="Yes",
    side="buy",
    price=0.55,
    size=100,
)
print(order.id, order.status)
```
</CodeGroup>
