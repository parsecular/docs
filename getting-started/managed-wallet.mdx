## Overview

Managed wallets are server-side wallets created by Parsec via Privy. You don't
need to manage private keys — Parsec signs transactions on your behalf using
a session signer.

## Step 1: Get your API key

Sign up at [parsecapi.com](https://parsecapi.com) → **Settings** → copy your API key (`pk_live_...`).

## Step 2: Register a wallet

<CodeGroup>
```bash curl
curl -X POST "https://api.parsecapi.com/api/v1/wallet" \
  -H "X-API-Key: pk_live_YOUR_KEY"
```
```typescript TypeScript
import ParsecAPI from 'parsec-api';

const client = new ParsecAPI({ apiKey: 'pk_live_YOUR_KEY' });

const wallet = await client.wallet.register();
console.log(wallet.eoa_address);
// "0x1234...abcd"
```
```python Python
from parsec_api import ParsecAPI

client = ParsecAPI(api_key="pk_live_YOUR_KEY")

wallet = client.wallet.register()
print(wallet.eoa_address)
# "0x1234...abcd"
```
</CodeGroup>

<ResponseExample>
```json
{
  "eoa_address": "0x1234567890abcdef1234567890abcdef12345678",
  "privy_wallet_id": "wallet_abc123"
}
```
</ResponseExample>

<Info>
Idempotent — calling this again returns your existing wallet.
</Info>

## Step 3: Fund your wallet

Send **USDC** and a small amount of **POL** (for gas) to your `eoa_address` on **Polygon**.

| Token | Amount | Purpose |
|-------|--------|---------|
| USDC | Your trading capital | Collateral for orders |
| POL | ~0.5 POL | Gas for approval transactions |

## Step 4: Link to Polymarket

This single call handles everything needed to trade on Polymarket:
1. Grants a session signer to your wallet
2. Derives CLOB API credentials
3. Sets token approvals (USDC, CTF, Neg Risk)
4. Stores credentials for future use

<CodeGroup>
```bash curl
curl -X POST "https://api.parsecapi.com/api/v1/link?exchange=polymarket" \
  -H "X-API-Key: pk_live_YOUR_KEY"
```
```typescript TypeScript
const link = await client.link.create({ exchange: 'polymarket' });
console.log(link.status, link.linked_exchanges);
```
```python Python
link = client.link.create(exchange="polymarket")
print(link.status, link.linked_exchanges)
```
</CodeGroup>

<ResponseExample>
```json
{
  "exchange": "polymarket",
  "status": "linked",
  "wallet_address": "0x1234567890abcdef1234567890abcdef12345678",
  "linked_exchanges": ["polymarket"]
}
```
</ResponseExample>

## Step 5: Check your wallet state

<CodeGroup>
```bash curl
curl -H "X-API-Key: pk_live_YOUR_KEY" \
  "https://api.parsecapi.com/api/v1/wallet"
```
```typescript TypeScript
const state = await client.wallet.get();
console.log(state);
```
```python Python
state = client.wallet.get()
print(state)
```
</CodeGroup>

<ResponseExample>
```json
{
  "wallet": {
    "eoa_address": "0x1234...abcd",
    "privy_wallet_id": "wallet_abc123",
    "created_at": "2026-01-15T12:00:00Z"
  },
  "session_signer": {
    "active": true,
    "expires_at": "2026-02-15T12:00:00Z",
    "signer_id": "signer_xyz"
  },
  "linked_exchanges": [
    { "exchange": "polymarket", "has_credentials": true }
  ]
}
```
</ResponseExample>

## Step 6: Trade

Place orders the same way as a direct trader — your managed wallet handles signing automatically.

<CodeGroup>
```typescript TypeScript
const order = await client.orders.create({
  exchange: 'polymarket',
  market_id: '0x1234abcd',
  outcome: 'Yes',
  side: 'buy',
  price: 0.55,
  size: 100,
});
console.log(order.id, order.status);
```
```python Python
order = client.orders.create(
    exchange="polymarket",
    market_id="0x1234abcd",
    outcome="Yes",
    side="buy",
    price=0.55,
    size=100,
)
print(order.id, order.status)
```
</CodeGroup>
