---
title: "Get Wallet"
description: "Get wallet state, session signer, and linked exchanges"
---

`GET /api/v1/wallet`

Returns the current state of your Parsec-managed wallet, including the session signer status and linked exchanges.

## Parameters

No parameters required.

## Response

<ResponseField name="wallet" type="object | null">
  Primary EOA wallet (backward-compatible). `null` if no wallet exists.
  <Expandable title="properties">
    <ResponseField name="eoa_address" type="string">
      EOA wallet address.
    </ResponseField>
    <ResponseField name="privy_wallet_id" type="string">
      Privy wallet identifier.
    </ResponseField>
    <ResponseField name="safe_address" type="string | null">
      Safe wallet address. Only present for `wallet_type: "safe"`.
    </ResponseField>
    <ResponseField name="wallet_type" type="string">
      `"eoa"` or `"safe"`.
    </ResponseField>
    <ResponseField name="chain_id" type="number | null">
      Chain ID (e.g., 137 for Polygon).
    </ResponseField>
    <ResponseField name="created_at" type="string | null">
      RFC 3339 creation timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="wallets" type="object[]">
  All wallets (EOA + Safe) associated with this account.
  Each element has the same shape as the `wallet` field above.
</ResponseField>

<ResponseField name="session_signer" type="object | null">
  Active session signer for order signing. `null` if no signer exists.
  <Expandable title="properties">
    <ResponseField name="active" type="boolean">
      Whether the session signer is currently active (not revoked).
    </ResponseField>
    <ResponseField name="expires_at" type="string | null">
      RFC 3339 expiration timestamp.
    </ResponseField>
    <ResponseField name="signer_id" type="string | null">
      Privy session signer identifier.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="linked_exchanges" type="object[]">
  Exchanges with stored credentials.
  <Expandable title="properties">
    <ResponseField name="exchange" type="string">
      Exchange identifier (e.g., `"polymarket"`).
    </ResponseField>
    <ResponseField name="has_credentials" type="boolean">
      Whether credentials are stored for this exchange.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash curl
curl "https://api.parsecapi.com/api/v1/wallet" \
  -H "X-API-Key: YOUR_API_KEY"
```
```typescript TypeScript
const state = await client.wallet.get();
console.log(state.wallet?.eoa_address);
console.log(state.linked_exchanges);
```
```python Python
state = client.wallet.get()
print(state.wallet.eoa_address if state.wallet else "No wallet")
print(state.linked_exchanges)
```
</RequestExample>
