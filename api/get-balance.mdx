---
title: "Get Balance"
description: "Get your account balance"
---

`GET /api/v1/balance`

Returns the raw balance response from the specified exchange. Response format varies by exchange.

## Parameters

<ParamField query="exchange" type="string" required>
  Exchange identifier: `polymarket`, `kalshi`, `limitless`, `opinion`, or `predictfun`.
</ParamField>

<ParamField query="refresh" type="boolean" default="false">
  Force a live balance fetch from the exchange, bypassing any cache.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request succeeded.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="raw" type="object">
      Raw JSON response from the exchange API. Structure varies by exchange.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecapi.com/api/v1/balance?exchange=polymarket"

# Force live fetch
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecapi.com/api/v1/balance?exchange=kalshi&refresh=true"
```
</RequestExample>

<ResponseExample>
```json Polymarket
{
  "success": true,
  "data": {
    "raw": {
      "balance": "14328266",
      "allowance": "115792..."
    }
  }
}
```
</ResponseExample>

<ResponseExample>
```json Kalshi
{
  "success": true,
  "data": {
    "raw": {
      "balance": 1000
    }
  }
}
```
</ResponseExample>

## Exchange-Specific Notes

| Exchange | Balance Field | Unit |
|----------|--------------|------|
| Polymarket | `balance` | Micro-USDC (divide by 1,000,000) |
| Kalshi | `balance` | Cents (divide by 100) |
| Limitless | `currentValue` | USD (PnL) |
