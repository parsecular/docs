---
title: "List Exchanges"
description: "Get available exchanges"
---

`GET /api/v1/exchanges`

Returns exchange capabilities available to the authenticated user.

## Parameters

No parameters required.

## Response

<ResponseField name="(root)" type="ExchangeCapability[]">
  Array of capability objects with per-exchange support flags.
</ResponseField>

<RequestExample>
```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecapi.com/api/v1/exchanges"
```
</RequestExample>

<ResponseExample>
```json
[
  {
    "id": "polymarket",
    "name": "Polymarket",
    "has": {
      "fetch_markets": true,
      "create_order": true,
      "websocket": true
    }
  },
  {
    "id": "opinion",
    "name": "Opinion",
    "has": {
      "fetch_markets": true,
      "create_order": false,
      "websocket": false
    }
  }
]
```
</ResponseExample>
