---
title: "List Exchanges"
description: "Get available exchanges"
---

`GET /api/v1/exchanges`

Returns exchanges available to the authenticated user.

## Parameters

No parameters required.

## Response

<ResponseField name="success" type="boolean">
  Whether the request succeeded.
</ResponseField>

<ResponseField name="data" type="string[]">
  Array of exchange identifiers.
</ResponseField>

<RequestExample>
```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecular.com/api/v1/exchanges"
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": ["polymarket", "kalshi", "limitless", "opinion", "predictfun"]
}
```
</ResponseExample>
