---
title: "List Exchanges"
description: "Get available exchanges"
---

`GET /api/v1/exchanges`

Returns exchanges available to the authenticated user.

## Parameters

No parameters required.

## Response

<ResponseField name="(root)" type="string[]">
  Array of exchange identifiers.
</ResponseField>

<RequestExample>
```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecapi.com/api/v1/exchanges"
```
</RequestExample>

<ResponseExample>
```json
["polymarket", "kalshi", "limitless", "opinion", "predictfun"]
```
</ResponseExample>
