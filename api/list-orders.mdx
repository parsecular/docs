---
title: "List Orders"
description: "Get open orders for an exchange"
---

`GET /api/v1/orders`

Returns open orders for the authenticated user on the specified exchange.

## Error Semantics

- `403` means credentials/authorization are missing or invalid for private exchange access.
- `503` means the upstream exchange is temporarily unavailable (retry with backoff).

## Parameters

<ParamField query="exchange" type="string" required>
  Exchange identifier: `polymarket`, `kalshi`, `limitless`, `opinion`, or `predictfun`.
</ParamField>

<ParamField query="market_id" type="string">
  Filter orders by a specific market ID.
</ParamField>

## Response

<ResponseField name="(root)" type="Order[]">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Order identifier.
    </ResponseField>
    <ResponseField name="market_id" type="string">
      Market the order is on.
    </ResponseField>
    <ResponseField name="outcome" type="string">
      The outcome being traded.
    </ResponseField>
    <ResponseField name="side" type="string">
      `"buy"` or `"sell"`.
    </ResponseField>
    <ResponseField name="price" type="number">
      Order price.
    </ResponseField>
    <ResponseField name="size" type="number">
      Total order size.
    </ResponseField>
    <ResponseField name="filled" type="number">
      Amount filled so far.
    </ResponseField>
    <ResponseField name="status" type="string">
      One of: `pending`, `open`, `filled`, `partially_filled`, `cancelled`, `rejected`.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      ISO 8601 creation timestamp.
    </ResponseField>
    <ResponseField name="updated_at" type="string | null">
      ISO 8601 last update timestamp.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
# All open orders
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecapi.com/api/v1/orders?exchange=polymarket"

# Filtered by market
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.parsecapi.com/api/v1/orders?exchange=polymarket&market_id=0x1234abcd"
```
</RequestExample>
