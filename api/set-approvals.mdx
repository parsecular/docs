---
title: "Set Approvals"
description: "Set token approvals for Polymarket trading"
---

`POST /api/v1/approvals`

Submit on-chain transactions to approve Polymarket contracts to spend your USDC and CTF tokens.

<Note>
This endpoint only supports Polymarket.
</Note>

<Warning>
Submits blockchain transactions on Polygon. Ensure your wallet has sufficient MATIC for gas fees.
</Warning>

## Parameters

<ParamField query="exchange" type="string" required>
  Must be `polymarket`.
</ParamField>

### Body

<ParamField body="all" type="boolean" default="false">
  Approve all tokens for all contracts.
</ParamField>

<ParamField body="usdc" type="boolean" default="false">
  Approve USDC for all contracts.
</ParamField>

<ParamField body="ctf" type="boolean" default="false">
  Approve CTF for all contracts.
</ParamField>

<ParamField body="usdc_neg_risk" type="boolean" default="false">
  Approve USDC for negative risk contracts only.
</ParamField>

<ParamField body="ctf_neg_risk" type="boolean" default="false">
  Approve CTF for negative risk contracts only.
</ParamField>

## Response

<ResponseField name="(root)" type="object">
  <Expandable title="properties">
    <ResponseField name="results" type="array">
      Array of approval results.
      <Expandable title="ApprovalResult properties">
        <ResponseField name="token" type="string">
          Token type: `usdc` or `ctf`.
        </ResponseField>
        <ResponseField name="target" type="string">
          Target contract.
        </ResponseField>
        <ResponseField name="tx_hash" type="string | null">
          Transaction hash if successful.
        </ResponseField>
        <ResponseField name="success" type="boolean">
          Whether this approval succeeded.
        </ResponseField>
        <ResponseField name="error" type="string">
          Error message, omitted on success.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="all_succeeded" type="boolean">
      Whether all transactions succeeded.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
# Approve all tokens
curl -X POST \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"all": true}' \
  "https://api.parsecapi.com/api/v1/approvals?exchange=polymarket"

# Approve USDC only
curl -X POST \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"usdc": true}' \
  "https://api.parsecapi.com/api/v1/approvals?exchange=polymarket"
```
</RequestExample>
