---
title: "Update Credentials"
description: "Update exchange credentials for your API key"
---

`PUT /api/v1/credentials`

Updates the stored exchange credentials for the authenticated API key. Returns 204 on success.

All fields are optional; only include the credentials you want to update. Fields set to `null` are ignored.

## Parameters

### Kalshi

<ParamField body="api_key_id" type="string | null">
  Kalshi API key ID from the Kalshi dashboard.
</ParamField>

<ParamField body="private_key" type="string | null">
  Kalshi RSA private key (PEM format) for request signing.
</ParamField>

### Polymarket

<ParamField body="clob_api_key" type="string | null">
  Polymarket CLOB API key.
</ParamField>

<ParamField body="clob_api_secret" type="string | null">
  Polymarket CLOB API secret.
</ParamField>

<ParamField body="clob_api_passphrase" type="string | null">
  Polymarket CLOB API passphrase.
</ParamField>

## Response

Returns `204 No Content` on success with no body.

<RequestExample>
```bash Kalshi
curl -X PUT "https://api.parsecapi.com/api/v1/credentials" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "api_key_id": "your-kalshi-key-id",
    "private_key": "-----BEGIN RSA PRIVATE KEY-----\nMIIE..."
  }'
```
```bash Polymarket
curl -X PUT "https://api.parsecapi.com/api/v1/credentials" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "clob_api_key": "your-clob-api-key",
    "clob_api_secret": "your-clob-api-secret",
    "clob_api_passphrase": "your-clob-passphrase"
  }'
```
</RequestExample>
