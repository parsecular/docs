`POST /api/v1/link`

Links your Parsec-managed wallet to an exchange. Currently supports Polymarket.

This single call:
1. Grants a session signer to your wallet
2. Derives CLOB API credentials
3. Sets token approvals (USDC, CTF, Neg Risk)
4. Stores credentials for future use

<Warning>
Requires wallet registration first â€” call `POST /api/v1/wallet` before linking.
</Warning>

## Parameters

<ParamField query="exchange" type="string" required>
  Exchange to link. Currently only `polymarket` is supported.
</ParamField>

## Response

<ResponseField name="exchange" type="string">
  The exchange that was linked.
</ResponseField>

<ResponseField name="status" type="string">
  Link status (e.g., `"linked"`).
</ResponseField>

<ResponseField name="wallet_address" type="string">
  The wallet address used for the link.
</ResponseField>

<ResponseField name="linked_exchanges" type="array">
  Updated list of all linked exchange identifiers.
</ResponseField>

<RequestExample>
```bash curl
curl -X POST "https://api.parsecapi.com/api/v1/link?exchange=polymarket" \
  -H "X-API-Key: YOUR_API_KEY"
```
```typescript TypeScript
const link = await client.link.create({ exchange: 'polymarket' });
console.log(link.status); // "linked"
```
```python Python
link = client.link.create(exchange="polymarket")
print(link.status)  # "linked"
```
</RequestExample>
