## What is Parsec?

One API, every prediction market. Parsec gives you a unified REST API and
WebSocket feed across Polymarket, Kalshi, Limitless, Opinion, and Predict.fun.

## Quick start

Install the SDK and fetch markets in 3 lines.

<CodeGroup>
```bash npm
npm install parsec-api
```
```bash pip
pip install parsec_api
```
</CodeGroup>

**Fetch markets:**

<CodeGroup>
```typescript TypeScript
import ParsecAPI from 'parsec-api';

const client = new ParsecAPI();

const markets = await client.markets.list({ exchanges: ['polymarket'], limit: 5 });
console.log(markets.markets);
```
```python Python
from parsec_api import ParsecAPI

client = ParsecAPI()

markets = client.markets.list(exchanges=["polymarket"], limit=5)
print(markets.markets)
```
</CodeGroup>

**Place an order:**

<CodeGroup>
```typescript TypeScript
const order = await client.orders.create({
  exchange: 'polymarket',
  market_id: '0x...',
  outcome: 'yes',
  side: 'buy',
  price: 0.55,
  size: 100,
});
console.log(order.id, order.status);
```
```python Python
order = client.orders.create(
    exchange="polymarket",
    market_id="0x...",
    outcome="yes",
    side="buy",
    price=0.55,
    size=100,
)
print(order.id, order.status)
```
</CodeGroup>

**Stream real-time orderbooks:**

<CodeGroup>
```typescript TypeScript
const ws = client.ws();

ws.on('orderbook', (book) => {
  console.log(`${book.parsecId}: best bid ${book.bids[0]?.price}`);
});

await ws.connect();
ws.subscribe({ parsecId: 'polymarket:572473', outcome: 'Yes' });
```
```python Python
import asyncio
from parsec_api import ParsecAPI

client = ParsecAPI()
ws = client.ws()

@ws.on("orderbook")
async def on_book(book):
    print(f"{book.parsec_id}: best bid {book.bids[0].price if book.bids else 'N/A'}")

async def main():
    await ws.connect()
    ws.subscribe(parsec_id="polymarket:572473", outcome="Yes")
    await ws.run_forever()

asyncio.run(main())
```
</CodeGroup>
