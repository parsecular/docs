## Overview

Parsecular is a unified API layer for prediction markets, designed with the same philosophy as CCXT for cryptocurrency exchanges. It provides a consistent, type-safe interface for interacting with multiple prediction market platforms through a single API.

## Supported Exchanges

Parsecular supports five major prediction market platforms:

| Exchange | REST API | WebSocket | Auth Type |
|----------|----------|-----------|-----------|
| **Polymarket** | Full | Orderbook | Ethereum Wallet |
| **Limitless** | Full | Orderbook | Ethereum Wallet |
| **Kalshi** | Full | - | RSA Certificate |
| **Opinion** | Full | - | API Key + Multi-sig |
| **Predict.fun** | Full | - | API Key + Ethereum |

## Quick Example

```rust
use pc_core::{Exchange, FetchMarketsParams};
use pc_exchange_polymarket::{Polymarket, PolymarketConfig};

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    // Initialize exchange
    let exchange = Polymarket::new(PolymarketConfig::new())?;

    // Fetch markets
    let markets = exchange
        .fetch_markets(Some(FetchMarketsParams {
            limit: Some(5),
            active_only: true,
        }))
        .await?;

    for market in markets {
        println!("{}: {}", market.id, market.question);
    }

    Ok(())
}
```

## Installation

Add Parsecular to your `Cargo.toml`:

```toml
[dependencies]
pc-core = { path = "pc-core" }
pc-exchange-polymarket = { path = "pc-exchange-polymarket" }
pc-exchange-limitless = { path = "pc-exchange-limitless" }
pc-exchange-kalshi = { path = "pc-exchange-kalshi" }
pc-exchange-opinion = { path = "pc-exchange-opinion" }
pc-exchange-predictfun = { path = "pc-exchange-predictfun" }

tokio = { version = "1", features = ["full"] }
anyhow = "1"
```
